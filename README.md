| [Windows 10 Client](https://www.microsoft.com/windows/) | Version 1607+(**\***)      |
| [Windows-11](https://www.microsoft.com/windows/)        | 2008 R2 SP1(**\***), 2012+ |

| OS                                       | Version |

- [Oracle JavaSE8](https://www.oracle.com/cn/java/technologies/downloads/#java8)
